# WeatherAlert
[![Build Status](https://travis-ci.org/AlexKel/WeatherAlert.svg?branch=develop)](https://travis-ci.org/AlexKel/WeatherAlert)

### Requirements
* iOS 8.0
* Cocoapods v1.0.0

### Installation
* Pull development branch
* Run `pod install` in terminal
* Open with `WeatherAlert.xcworkspace`
